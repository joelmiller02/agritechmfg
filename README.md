# agritechmfg
AgriTech Manufacturing Website

## Git Usage
Go to the git tab in VS Code and sync before changes.

Test your changes with Live Server in the browser.

After changes, commit with a summary of changes.

Make sure to push your changes to Github.

## How to run a local testing server
Click the "Go Live" in the bottom right of VS Code
The links do not work correctly.

or

Right click each html file and click "Open With Live Server"
